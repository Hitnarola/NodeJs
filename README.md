# NodeJs

This repository contains code samples and notes for learning Node.js, including modules, events, and more.

## Folder Structure

- `01-modules/` — Examples and notes about Node.js modules and the require system
- `02-event/` — EventEmitter examples and custom event handling in Node.js

## Getting Started

1. Make sure you have [Node.js](https://nodejs.org/) installed.
2. Clone this repository or download the code.
3. Navigate to a folder and run example files using:
   ```sh
   node <filename.js>
   ```

## Example

To see how events work in Node.js:

```sh
cd 02-event
node myevents.js
```

## Learning Topics

- Node.js modules and require
- EventEmitter and custom events
- Basic Node.js architecture
